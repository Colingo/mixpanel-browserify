# mixpanel-browserify

<!-- [![build status][1]][2] [![dependency status][3]][4]

[![browser support][5]][6] -->

A browserify compliant version of mixpanel JS client

## Example

```js
var mixpanel = require("mixpanel-browserify")

mixpanel.init({ /* tokens */ })

mixpanel.doStuff()
```

## Installation

`npm install mixpanel-browserify`
